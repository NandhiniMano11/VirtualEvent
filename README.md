# Virtual Event 
## Install

    $ git clone https://github.com/NandhiniMano11/VirtualEvent.git
    $ cd VirtualEvent
    $ npm install

## Running the project
    
    $ npm start

## Running the swagger 
     Open browser 
     Browse http://localhost:4000/api-docs
